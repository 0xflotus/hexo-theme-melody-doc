## Features

* Render static pages fastly (Use the [pug](https://github.com/pugjs/pug) template engine)
* Smoothly update (Use the Hexo [data files](https://hexo.io/docs/data-files.html) feature)
* Comments support
* Share support
* Search support
* Analytics support
* Animation support
* Catalog support
* MathJax support
* i18n support
* Compatible with IE10+
* ...
